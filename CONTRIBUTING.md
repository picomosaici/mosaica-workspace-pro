# Come contribuire a Mosaica Workspace Pro

Grazie per il tuo interesse! 🎨

## 🐛 Segnalazione bug

Il modo più utile per contribuire è segnalare problemi che incontri usando Mosaica. Apri una [Issue](https://github.com/picomosaici/mosaica-workspace-pro/issues) descrivendo:

- **Cosa stavi facendo** quando si è verificato il problema
- **Cosa ti aspettavi** dovesse succedere
- **Cosa è successo invece**
- **Versione di Mosaica** (formato MM.YYYY, visibile nell'header della guida utente)
- **Versione di Windows** (10 o 11)
- Screenshot, se possibili

## 💡 Suggerimenti

Proposte di nuove funzionalità o miglioramenti sono sempre benvenute. Aprile come Issue con il prefisso `[Idea]` nel titolo.

## 🔧 Pull Request

Mosaica è un software proprietario: al momento non vengono accettate pull request esterne sul codice. Se hai una proposta tecnica importante contattami via email per discuterne (picomosaici@gmail.com).

## 📜 Codice di condotta

Sii cortese, rispettoso e costruttivo. Le discussioni tecniche sono benvenute, gli insulti no.
