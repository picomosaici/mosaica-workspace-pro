The Six Panels

① Top Bar (at the top)

Logo, File dropdown (Open/Save/New Project + auto-open), Export dropdown (PDF, PNG/SVG, freehand), Undo/Redo buttons, Zoom controls (+/−/reset with live percentage), Calibration dropdown, full screen toggle (⛶/🗗), 📖 Guide button, 🖊️ Wacom button (tablet settings with connection status indicator), and 🖱️⌨️ Input button (custom mouse and keyboard mapping).

② BG Panel (right below)

Horizontal bar dedicated to the background image: drop area, thumbnail preview, Contain/Cover buttons, rotation (↶ ↷), and removal.

③ Texture Panel (far left)

Narrow vertical column with the available texture grid. Click a thumbnail to apply the texture to the selected object.

④ Left Toolbar (after textures)

All tools: canvas rotation (Portrait/Landscape), Select (default), Add shape, Lasso, Pen, Eraser, Watercolor, Line list, Clear all.

⑤ Inspector (on the right)

Contextual panel that changes automatically based on the active tool or selected shape: pen/watercolor parameters, triangle angle sliders, trapezoid controls, dimensions in mm.

⑥ Status Bar (at the bottom)

Zoom %, cursor coordinates in mm, type and size of the selection, auto-save status, calibration factor.

Top Bar — File Dropdown

Clicking File in the top bar opens a menu with these entries:

  • 📂 Open Project… — opens an existing .msp.json file.
  • 💾 Save Project — saves the current project in .msp.json format.
  • New Project (purple button) — clears the canvas and creates a blank sheet with a small reference rectangle. Opens a confirmation modal first because this action deletes all unsaved work.
  • 🌐 Language — sub-section with two flag selectors 🇮🇹 Italiano and 🇬🇧 English. Clicking a flag switches the interface language and the app restarts automatically to apply translations to every module (toolbar, inspector, modals, user guide). The choice is persistent: it is saved in %APPDATA%\mosaica-workspace-pro\language.json and honored on every subsequent launch. The user guide follows the language too: in Italian it opens Guida_utente.html, in English Guida_utente_EN.html.
  • ☐ Open project at startup — checkbox to enable automatic opening of the last saved project (see dedicated section).
  • 🔄 Check for updates — checks the official GitHub repository for a newer version and, with your confirmation, downloads and installs it (see the Security & Updates section).

Top Bar — Export Dropdown

Clicking Export in the top bar opens a menu with three items, each opening a detailed choice modal:

  • 📄 Export PDF (A4) — 5 export modes, see "Export" section.
  • 🖼️ Export PNG / SVG — full PNG or vector SVG (filled or outlines only).
  • ✏️ Export freehand drawing — exports only freehand and watercolor strokes as SVG or PNG.

Status Bar — Reading the Numbers

The status bar at the bottom shows in real time:

  • Zoom — current zoom percentage (100% = 1:1 scale with calibration). Available range: 40%–800%.
  • Cursor — X, Y coordinates in real millimeters (zero = top-left corner of the A4 sheet).
  • Selection — type of the selected object + dimensions in real mm (e.g. rect · 25.4 × 18.0 mm), or the number of objects if it is a multi-selection.
  • Auto-save — appears only during an ongoing auto-save, then becomes "✔ Auto-saved" for 2 seconds.
  • Calibration — applied corrective scale factor (default 1.000).

Navigating the View

🖱️ Mouse Wheel

Cursor-centered zoom with a "Photoshop-style" system: during the gesture Mosaica takes a GPU snapshot of the canvas and scales it via CSS, so zooming stays fluid even on heavy projects (hundreds of tiles, large backgrounds, dozens of pen/watercolor strokes). The Fabric backstore is reallocated only once at the end of the gesture. Range 40%–800%: beyond these limits the value snaps to the extreme.

⌥ Alt + Drag

Hold Alt and drag to pan (move the view).

⌨️ Arrow Keys

With no object selected, arrows pan. With an object selected, they move the object (1 px, 10 px with Shift).

Adaptive Wheel Sensitivity

Wheel zoom has a sensitivity that adapts to gesture speed: when you turn the wheel slowly the zoom stays precise, one notch at a time; when you spin it quickly the zoom accelerates naturally to cover more range in less time, exactly like professional graphic editors. Speed is measured in events/second and smoothed with an exponential moving average (EMA) that decays over ~250 ms — so the moment you slow down, zoom returns to precise mode immediately. A safety cap on individual event intensity also protects against driver/OS spikes.

The GPU snapshot is capped at 4096 px on the longest side to keep RAM in check: at very high zoom levels you may notice a slight softening for a few milliseconds during the gesture, after which Fabric rasterizes at full quality. This is intentional, matching the behavior of Photoshop and Figma.
🎯
Practical Tips
  • Use Alt + drag to navigate the sheet without leaving the active tool; the cursor changes to "pan" automatically.
  • Wheel zoom is centered on the cursor: place the mouse exactly where you want to zoom in so you don't lose your working point.
  • If you are in pen, eraser, or watercolor mode and want to return to selecting shapes, click 🖱️ Select in the left toolbar: it automatically deactivates all modal tools.
  • The Status Bar shows coordinates in real mm: use it for precise positioning to the tenth of a millimeter. Coordinates disappear when the mouse leaves the workspace.
  • If you lose sight of the sheet, press Ctrl + 0 to return to 100% zoom and center the canvas.
  • The app starts in full screen mode: the 🗗 button at the top right (or ESC) exits it.

Screen Calibration

Mosaica works in real millimeters. To do this correctly, it needs to know how many pixels correspond to 1 millimeter on your screen. Calibration is done once per monitor.

Where to Find Calibration

In the top bar at the top right you will find the 📏 Calibration button with a numeric indicator next to it (e.g. 1.000). Clicking it opens a popover with all the necessary tools.

  1. Open the calibration panel
    Click 📏 Calibration in the top bar. A popover opens with a gray strip with a white border, exactly 50 mm theoretical length.
  2. Verify OS and browser zoom at 100%
    Before measuring, make sure the operating system zoom is at 100% (display scaling) and that no filters are active.
  3. Measure the strip with a caliper
    Place a digital caliper (or a rigid metal ruler) directly on the screen, on the strip labeled "50 mm THEORETICAL". Record the actual measurement.
  4. Enter the real measurement
    Type the measured value into the "Real measurement with caliper (mm)" field — e.g. 47.85 if the strip measures 47.85 mm.
  5. Click "Apply Calibration"
    The calibration factor is calculated, saved, and applied instantly. The indicator in the top bar and the one in the status bar update. All app measurements are recalibrated.
Calibration is automatically saved in the calibration.json file inside %APPDATA%\mosaica-workspace-pro\. There is no need to recalibrate every time you open the app, only if you change monitor or resolution.

The Reset calibration (1.0) button resets the factor to 1.000 (no correction).

🎯
Practical Tips
  • Use a digital caliper or a rigid physical ruler (metal or hard plastic) instead of a flexible tape measure: the reading is more precise to the hundredth of a millimeter.
  • If you change monitor, resolution, or screen orientation, recalibrate: the factor is tied to that specific display.
  • To verify calibration, double-click on the sheet to create a square and check that the dimensions shown in the Inspector and in the status bar match the measurement of the caliper placed on the screen.
  • The calibration factor indicator is always visible both in the top bar button and in the right side of the status bar: at a glance you know whether you are calibrated or have a corrective factor applied.

Canvas Rotation (Portrait ↔ Landscape)

The two buttons at the top of the Left Toolbar let you instantly switch between vertical (A4 portrait, 210 × 297 mm) and horizontal (A4 landscape, 297 × 210 mm) orientation, geometrically rotating the entire sheet content.

"Portrait" Button

Sets the canvas to A4 portrait format (210 × 297 mm).

"Landscape" Button

Sets the canvas to A4 landscape format (297 × 210 mm).

Rotation is geometric and precise:
• All shapes, freehand strokes, and watercolor are rotated.
• The paper texture is automatically recreated to the new dimensions.
• The background image is readjusted.
• The mm calibration remains unchanged.
• Undo/Redo works normally after rotation.

The current orientation is saved in the .msp.json project file and restored on opening. PDF export also respects the sheet orientation (portrait/landscape).

🎯
Practical Tips
  • Before rotating the canvas, position all shapes where you want them: rotation occurs around the geometric center of the A4 sheet.
  • After a rotation, check the background image: if you were using Contain or Cover, you may need to readjust the background rotation separately (using the ↶ Rotate buttons in the BG Panel).
  • The paper texture automatically regenerates with the correct orientation: you do not need to reload it manually.
  • Undo/Redo works normally even after an orientation change, so you can undo the rotation if the result is not satisfactory.

Creating and Managing Shapes

Four Ways to Create a Shape

  • Double-click on an empty area of the sheet → creates a ~10 mm square at the clicked point.
  • ➕ Add Shape button in the Left Toolbar → creates a ~10 mm square at the center of the visible canvas portion, independent of zoom and pan. Handy if you have zoomed far away and don't want to return to the sheet to insert a shape.
  • ✏️➕ Create Custom Shape button in the Left Toolbar → opens a dedicated modal where you draw a non-standard polygon by tracing its vertices click after click, modify them vertex by vertex, and then insert it into the canvas. See the dedicated paragraph below.
  • When starting a New Project a reference rectangle is created at the center of the sheet.
The rectangle is the starting shape for "regular" polygons. Use the 🔄 Change Shape button in the radial menu to transform it into a circle, trapezoid, triangle, or circle sector. For free shapes (stars, arrows, irregular silhouettes) use the ✏️➕ Create Custom Shape builder instead.

Selecting and Moving

Click a shape to select it. Blue control handles will appear at the edges and corners for resizing, and the ↻ symbol for rotation. Drag the shape to move it.

Multi-Selection

You have four ways:

  • Drag on an empty area of the sheet → classic selection box.
  • Ctrl + click (⌘ on Mac) on multiple shapes → adds or removes shapes one at a time: clicking a shape already in the selection removes it.
  • Lasso (see dedicated section) → trace a freehand polygon to enclose an area of shapes.
  • Brush selection (see dedicated section) → paint over the tiles to gather them along a line or an edge.

Measurements in Millimeters

When a shape is selected, two labels appear above it with the real measurements (Width × Height) in mm. Clicking on the W or H label lets you manually enter the desired dimension: type the value, press Enter to apply or Esc to cancel.

In addition, when a single shape is selected, at the top of the Inspector on the right you see the Dimensions block with two fields Width and Height in mm, always editable.

Shape Cycle

The 🔄 Change Shape button in the radial menu cycles through:

Rectangle Circle / Ellipse Trapezoid Isosceles Triangle Scalene Triangle Equilateral Triangle Right Triangle Acute Triangle Obtuse Triangle

Inspector — Triangle (Angle Sliders)

When you select a triangle, the Inspector automatically shows the "Triangle · Angles" section with two sliders:

  • α (alpha) — bottom-left angle, 1°–178°
  • β (beta) — bottom-right angle, 1°–178°

The third angle γ is calculated automatically (sum = 180°). The visual dimensions of the triangle are preserved while you change the angles. Snap is 0.1°.

Inspector — Trapezoid

When you select a trapezoid, the Inspector shows the dedicated section with sliders for:

  • Top base (in mm)
  • Bottom base (in mm)
  • Height (in mm)
  • Offset — horizontal shift of the top base relative to the center (changes the "category": rectangle, isosceles, scalene)

Divide Circle into Sectors

Select a circle or ellipse, then use the ◔ Divide button in the radial menu:

  • Normal click → divides into 4 quarters (90° each)
  • Alt + Click → divides into 2 halves (180° each)

Each sector becomes an independent object, resizable and modifiable.

Custom Shapes (Non-Standard)

When regular shapes (rectangle, circle, triangle, trapezoid, sector) are not enough — for example for stars, arrows, petals, irregular silhouettes, irregular tile outlines for your mosaics — use the custom shape builder from the Left Toolbar (button ✏️➕ right next to ➕ Add Shape).

The builder opens a dedicated modal with a preview area equipped with real millimeter rulers (top and left), zoom and pan independent from the main canvas, and works in two phases: first you trace the vertices, then you freely modify vertices and the curvature of individual sides before inserting it into the canvas.

The drawing area: rulers, zoom, pan

  • Real mm rulers at the top and left: they use the same calibration as the main canvas, so the millimeters you see in the builder correspond exactly to those on the A4 sheet. Major ticks every 10 mm (labeled), medium ticks every 5 mm, fine ticks every 1 mm (visible when zoom is high enough). Label density auto-regulates based on zoom.
  • Drawing area zoom: mouse wheel for cursor-centered zoom, or the − / + / ⟲ buttons at the top right. Range 25% – 800% (wider than the main canvas, because fine precision on handles is often needed here). The percentage label is always visible next to the buttons.
  • Drawing area pan: Alt + left-button drag (same logic as the main canvas). The middle mouse button (pressed wheel) also works for pan.
  • Cursor coordinates in mm: at the bottom right of the drawing area there is a live indicator x: … mm y: … mm showing the cursor position in real millimeters — useful for drawing to scale.

Phase 1 — Tracing Sides (Drawing)

  • 1st click on the preview area → sets the first vertex. From that moment a dashed blue line follows the mouse without needing to hold down, extending and tilting freely.
  • 2nd click → sets the end of the first line. The second line automatically starts from the last vertex and follows the mouse.
  • 3rd click, 4th click… Nth click → each click adds a vertex and starts the next side. You can place as many vertices as you want.
  • Backspace (or ↶ Vertex button) → removes the last placed vertex if you made a mistake.
  • 🔄 Restart → clears everything and starts over (also resets zoom).

Closing the Shape

With at least 3 vertices you can close the polygon in two ways:

  • Move the mouse close to the first vertex (the green one): when you enter the snap radius the first vertex enlarges and the ghost line becomes solid green. Click → the shape closes.
  • Press Enter → the shape closes by joining the last vertex to the first, regardless of where the mouse is.

Phase 2 — Editing Vertices and Curving Sides (Editing)

As soon as the shape is closed, a solid blue dot handle appears on every vertex and a small semi-transparent round blue handle at the center appears on every side. For example a six-pointed star will have six vertex handles and six side handles.

Moving a vertex:

  • Click and drag a vertex handle → the corresponding vertex moves, and the two sides forming it lengthen and tilt in real time until you release the mouse. If one of the two sides was curved, the curvature follows the vertex (the curve deforms accordingly).

Curving a single side:

  • Click and drag the round blue handle at the center of the side → the side transforms into a quadratic Bézier curve that passes exactly through the point where you are dragging the handle. The further you drag from the original straight line, the more pronounced the curve becomes.
  • When a side is curved, its handle becomes solid orange to distinguish it at a glance from the handles of still-straight sides.
  • You can curve as many sides as you want, even all of them, each independently. Every curve is stored separately — modifying the curvature of one side does not alter the others.
  • Even the closing side (the one joining the last vertex to the first) is curvable like all the others.

Resetting a curved side to straight:

  • Double-click on the orange handle of the curved side → the side immediately becomes straight again.
  • Right-click on the orange handle → same effect (alternative for those who prefer right-click). The browser's native context menu is not shown.
You can freely alternate between moving vertices and curving sides until the shape satisfies you. The shape remains in editing phase until you insert it or cancel, and the drawing area zoom/pan remain active even during editing — so you can zoom to 400% to precisely tweak a small curve and then return to full view with ⟲ or by pressing 0.

Insert into Canvas

When the shape satisfies you, press ✅ Insert into Canvas (or Enter if you are already in editing phase): the modal closes and the shape is placed at the center of the visible canvas portion — exactly with the same placement logic as the ➕ Add Shape button, independent of the main canvas zoom and pan. The shape maintains the preview dimensions at 1:1 real mm scale.

Once in the canvas, the custom shape behaves exactly like all the others: radial menu, blue handles for resizing and rotating, mm measurements in the Inspector and status bar, application of color and texture, duplication, group/ungroup, lasso selection, undo/redo, and is saved in the .msp.json project along with everything else — including any side curves.

Working sheet: formats and units

At the top of the modal you choose the sheet format you draw on: the presets A5, A4, A3, A2, A1, A0 or Custom with width and height typed by hand. The field unit can be switched between mm, cm and inches (conversion is automatic). Next to it you always see the sheet size and area updated in real time; rulers and grid adapt to the new format immediately.

⚠️
A shape can be inserted into the main canvas only if the builder sheet is A4 or smaller: Mosaica's canvas is a real A4. On larger sheets (A3→A0) the builder is for designing and saving the composition (see "Multiple shapes together"), but the Insert button stays disabled until you go back to a format ≤ A4.

Shape color and opacity

The builder toolbar has a color picker and an opacity slider (0–100%) that act on the active shape. The last combination you choose becomes the default inherited by the shapes you create afterwards, so you can draw a consistent set without resetting it every time.

Rounding corners — global and per single vertex

  • All corners — the "All corners" slider (in mm) rounds every vertex of the active shape with the same radius.
  • Selected corner — in the editing phase click a single vertex to select it, then use the "Selected corner" slider to give it a radius independent of the others. Each vertex remembers its own radius: you can have a shape with a single soft corner and all the others sharp.
The "All corners" slider takes precedence: moving it overrides the per-vertex radii set by hand. It is best to set the global rounding first and then fine-tune individual vertices.

Multiple shapes together — "palladiana" projects

In the builder you can draw several independent shapes on the same sheet ("software within the software"), designed to lay out a palladiana pattern or an irregular-tile mosaic before bringing it into the canvas.

  • ➕ New shape — closes the current shape and starts a new empty one, inheriting the default color, opacity and rounding.
  • Each closed shape can be moved by dragging its body, reselected to edit its vertices and curves, or deleted with 🗑 Delete shape.
  • 💾 Save palladiana / 📂 Open palladiana — save and reload the entire builder project (all shapes with their vertices, curves, radii, colors, plus the sheet format) to a dedicated file, so you can resume the composition later.

Reference image (tracing)

You can load a reference image into the builder to trace it with the vertices: the image is drawn under the grid and clipped to the sheet. Entering "image edit mode" you move and scale it with the handles; leaving that mode you go back to tracing vertices over it.

The grid adapts its own color to the image automatically: over light images it turns dark (blue, distinguishable from black strokes), over dark images it turns light (cyan), so the vertices always stay readable on top of the reference.

Keyboard Shortcuts Inside the Modal

  • Enter → in drawing closes the shape (with ≥3 vertices); in editing inserts the shape into the canvas.
  • Backspace / Delete → removes the last vertex (drawing only).
  • + → zoom in of the drawing area (centered on the preview center).
  • − → zoom out of the drawing area.
  • 0 → resets zoom to 100% and re-centers pan.
  • Esc → closes the modal without inserting anything.
⚠️
The builder does not check whether sides self-intersect: if sides cross (straight or curved) you will get a "butterfly" polygon with fill according to the Fabric even-odd rule. This is often a desired feature (shapes like hourglasses, infinities, overlapping petals), but if it was not your intention use ↶ Vertex to go back, or double-click / right-click on the offending curve to straighten it, or 🔄 Restart.
🎯
Practical Tips
  • After creating a rectangle, use 🔄 Change Shape in the radial menu to immediately transform it into a circle, trapezoid, or triangle preserving position, color, and dimensions.
  • For precise dimensions to the tenth of a millimeter use the Width/Height fields in the Inspector: type the value and press Enter.
  • For triangles, modify the α and β angles from the Inspector first: the shape automatically preserves visual dimensions while you change the geometry.
  • For trapezoids, the Offset slider changes the "category" of the shape; the base sliders preserve the existing angles instead.
  • After resizing a shape with the blue handles, the system automatically bakes the scale into the points: the mm fields always remain consistent with what you see on screen.
  • To divide a circle into independent sectors, select it and use ◔ Divide: normal click for 4 quarters, Alt + click for 2 halves.
  • For irregular shapes (stars, arrows, petals, tile silhouettes) use the ✏️➕ Create Custom Shape builder: the mm rulers around the drawing area and the real-time cursor position indicator let you draw to scale, while dedicated zoom and pan give you the precision you need on the handles. After closing the shape you can curve individual sides by dragging the round blue handle at their center — useful for petals, leaves, waves, and any silhouette with soft edges. Once inserted into the canvas, you can always resize it with the blue handles or type the mm measurements in the Inspector.

Lasso Selection (Freehand Polygon)

The Lasso is an advanced selection method that lets you enclose shapes with a freehand drawn polygon. It is designed especially for mosaics, where tiles are often small, dense, and have tight joints: with a rectangular box you would catch too much or too little, while with the lasso you follow the real contour of the area you are interested in.

How to Activate

  1. Click the Lasso icon in the Left Toolbar
    The dashed oval icon with the "tail" activates lasso mode. If pen, eraser, or watercolor were active, they are automatically deactivated to avoid conflicts.
  2. Drag on the canvas to draw
    Hold the left button and move the mouse: a freehand polygon is traced, even with re-entrances or self-intersections. The preview is a dashed blue line with a faint azure fill.
  3. Release the left button
    All shapes whose internal portion exceeds the sensitivity threshold (see below) are selected in a native Fabric multi-selection. The tool auto-deactivates: you return to normal Select mode.
To exit the lasso mid-trace without confirming anything, press Esc: the polygon is cancelled and the tool deactivates.

How It Decides What to Include — The Algorithm

Unlike a "naive" lasso that only looks at the geometric center (useless on mosaics with small tiles), Mosaica uses a two-phase approach calibrated for dense mosaics:

  1. Broad-phase (AABB): for each shape the oriented bounding box is calculated and compared with the lasso's bounding box. Shapes clearly outside the area are immediately discarded (a very fast operation even with thousands of tiles).
  2. Narrow-phase (sampling): for the remaining shapes, multiple representative points are sampled — the center, the 4 corners of the oriented bounding box (which follows rotation and scale), the 4 midpoints of the sides, and for polygons / triangles also a sub-sample of the real transformed vertices. The percentage of these points falling inside the lasso polygon is measured and compared with the containment threshold.

The "point inside polygon" test is the classic ray casting, so it works correctly even with concave lassos or self-intersections.

Sensitivity (Containment Threshold)

Lasso sensitivity is adjustable and persistent. The factory value is 0.20 (20%), meaning it is enough that at least 20% of the sampled points of a tile fall inside the trace for the tile to be included. This default is tuned specifically for real mosaics with tiles ≤10–15 mm² and 2–3 mm joints, where tracing freehand between joints it is practically impossible not to brush against neighboring tiles.

  1. Open the settings popover
    Right-click on the Lasso button in the Left Toolbar (left click activates/deactivates the tool, right click opens the settings). A small popover titled 🎯 Lasso Sensitivity opens.
  2. Move the slider
    Range from 0.05 (5% — very permissive) to 1.00 (100% — only shapes almost entirely inside). The value applies in real time while you drag. Next to the slider you see the current value both as a decimal and as a percentage.
  3. Release to save
    At the end of the drag the value is written to calibration.json (field lassoContainmentThreshold) inside %APPDATA%\mosaica-workspace-pro\. It remains valid for all future sessions.

The Default (0.20) button in the popover resets sensitivity to the recommended factory value. Press Esc or click outside the popover to close it without additional changes.

The threshold is merged with the other fields in calibration.json (such as calibrationFactor): adjusting lasso sensitivity does not touch screen calibration, and vice versa.

After releasing the lasso you can:

  • Move all selected shapes together.
  • Open the radial menu and use ⎇ Group to create a permanent group.
  • Press Delete to delete them all.
  • Click outside the selection to deselect.
🎯
Practical Tips
  • Dense tiles and tight joints? Keep the threshold low (0.10–0.20). Even brushing against neighboring tiles, only those truly inside the area will be selected because the sampling evaluates multiple points, not just the center.
  • Large and well-spaced tiles? Raise the threshold (0.40–0.60) to be stricter: only tiles actually embraced by the lasso enter the selection.
  • Surgical selection of a single tile among others? High threshold (0.70–0.90) and trace tightly around the target tile: "brushes" against neighbors are ignored.
  • The tool auto-deactivates on release. If you need to do multiple consecutive lassos, click the icon again each time. This choice prevents staying in lasso mode without noticing and "eating" a normal selection click.
  • The lasso is perfect for selecting tiles arranged in spirals, along curved edges, or irregularly shaped areas where a rectangular box would catch too much.
  • Locked shapes (🔒), the user background, and the paper texture are always ignored: lock tiles already in place before a large lasso to avoid moving them accidentally.
  • During the lasso the cursor becomes a crosshair and Fabric's native rectangular selection is temporarily suspended, so the two selection methods do not confuse each other.

Brush selection (paint to select)

Next to the Lasso, the Left Toolbar has the 🖌️ Brush selection: instead of closing an area like the Lasso, here you "paint" directly over the tiles and every tile that ends up under the brush stripe gets selected. It is the most natural way to gather rows of tiles along a line, a curved edge or a grout joint, without having to close a polygon.

How to use it

  1. Activate Brush selection
    Click the 🖌️ icon in the Left Toolbar. If pen, eraser or watercolor were active they are turned off automatically. The cursor shows a preview circle as wide as the tip.
  2. Hold the left button and paint over the tiles
    The line follows the pointer exactly like the freehand pen. Every tile touched by the stripe is highlighted in real time. You can make several passes: strokes add up while the tool stays active.
  3. Confirm the selection
    Press Enter or click the brush icon again: the highlight becomes a real multi-selection and you return to Select mode. Esc cancels everything without selecting anything.
To remove tiles from a selection in progress, hold Ctrl (⌘ on Mac) while you paint: in that stroke the brush deselects instead of selecting. Handy for trimming the edges without starting over.

Tip size

The width of the stripe decides how many tiles you catch per pass. Right-click the icon to open a small popover with the Tip slider, measured in real millimeters on the sheet (range 2–60 mm, default 12 mm). The value is persistent: it comes back as it was when you reopen. The Default (12 mm) button restores the recommended size.

🎯
Lasso or Brush?
  • Brush → when the tiles you need lie along a line or an edge (a row, a curve, the outline of a figure): you sweep over them and collect them.
  • Lasso → when you need to enclose a full area of tiles: close the polygon and take everything inside.
  • Locked shapes (🔒), the background and the paper texture are always ignored by the brush too.
  • While the brush is active the shapes do not move: you can zoom and pan freely between passes.

Auto-Save

Mosaica automatically saves your work in the background every minute and when you close the app. No interaction required: it happens on its own.

How It Works

Auto-save has two modes depending on the project state:

① Already Open Project

If you have opened a file (manually or via auto-open), auto-save overwrites in-place that file wherever it is: HDD, USB, SD. It does not create duplicates.

② New Unsaved Project

If you are working on a new sheet (never explicitly saved), auto-save creates a session file in the userData/projects/ folder with an Italian date+time name.

Session File Name

Session files have this format:

1 April 2026 - 15&30.msp.json

The separator between hour and minutes is & (ampersand) and not : because on Windows the colon is reserved and would truncate the name creating empty files.

Automatic Pruning

The app keeps a maximum of 10 session files. When it creates the eleventh, the oldest is automatically deleted. The current file is always preserved. Projects you have manually saved with custom names are never touched: pruning only acts on files matching the date+time pattern.

Frequency and Conditions

  • Every 60 seconds, but only if the canvas is "dirty" (modified since the last save).
  • Anti-spam: at least 5 seconds between one save and the next.
  • During project serialization, auto-save is blocked to avoid loops.

Indicator in the Status Bar

When an auto-save starts, in the status bar at the bottom "💾 Auto-save in progress…" briefly appears, then "✔ Auto-saved" in green for about 2 seconds. If the save fails (e.g. USB unplugged) a warning is shown.

Closing the App

When you close the app with the X, Alt+F4, or another standard method, Mosaica:

  1. Intercepts the closing
    The operating system is "convinced" to wait. A full-screen informational overlay opens.
  2. Performs auto-save
    Saves the current project (in-place or as a session file, as above). The overlay shows the saved path and a spinner.
  3. 4-second countdown
    After saving, a 4-second countdown starts, then the app closes automatically. You can speed it up by clicking Exit now.
  4. "Cancel closing" button
    If you clicked X by mistake, click Cancel closing in the overlay: the app stays open and you can continue working.
If the app crashes or the PC is shut down abruptly, reopen Mosaica and from the File → Open Project menu look in the %APPDATA%\mosaica-workspace-pro\projects\ folder for the session file with the most recent date and time: you will find your work up to the penultimate auto-save (maximum loss ≈ 60 seconds).
🎯
Practical Tips
  • If you want a meaningful name for the project, save it explicitly with File → Save Project: from that moment auto-save overwrites that file instead of creating a session one.
  • Session files remain in the userData folder even after closing: every 10 files the oldest is deleted. If you need to keep a particular one, copy it elsewhere or rename it (the app will no longer delete it because it does not match the date+time pattern).
  • When working on a project saved on a USB flash drive or SD card, auto-save overwrites the file directly there: do not unplug the media during work or you will see a "Folder not available" error.
  • The "Cancel closing" button is a safety net: if you close by mistake, you have 4 seconds (plus save time) to stop the closing.

Wacom Tablet — Pressure, Tilt, Pen Buttons

Mosaica integrates a complete system for Wacom graphic tablets (and any other tablet exposing PointerEvent.pressure). Pressure is true, point-by-point: every segment of the stroke stores its own pressure, is serialized in .msp.json projects, exported in SVG, and participates in Undo/Redo like any other Fabric object.

Pen Button Calibration — Why It's Needed

Different Wacom drivers map the side buttons to different PointerEvent.button values: some fire button=2 for the lower one, others emit a contextmenu event directly, others use button=4 or 5 for the upper one, and on certain drivers the upper button arrives only as auxclick. To avoid blind heuristics, Mosaica allows you to learn the code of your specific driver.

  1. Click "Calibrate LOWER button"
    The banner turns yellow: "⏳ Press the LOWER pen button now…" — you have 10 seconds.
  2. Physically press the lower button
    Mosaica captures the code (e.g. code 2, or synthetic contextmenu if the driver only fires contextmenu) and saves it in learnedLowerButtonCode.
  3. Repeat for the UPPER button
    Same procedure. If the upper button is not detected after a few attempts, your driver does not emit it to the browser (hardware/driver limit): in that case, configure the upper button directly in the official Wacom Preferences.

The "Reset calibration" button returns to the default heuristics.

Live Test Canvas

At the bottom of the modal there is a small test canvas: draw freely with the pen to see in real time:

  • Current pressure in percentage (0–100%)
  • Tilt (magnitude and angle) — or "tilt not available" if the pen does not support it
  • Stroke preview with current settings applied

The "Clear test" button cleans the canvas; "Restore defaults" resets all Wacom settings to factory values (after confirmation).

Wacom settings are saved in localStorage (key mosaica_wacom_settings) and persist across sessions. They are independent from .msp.json projects: changing computers you will have to reconfigure your preferred sensitivity.
🎯
Practical Tips
  • If you have a Bamboo CTL-460, tilt does not work (sensors missing): disable Modulate tilt to avoid any unnecessary overhead, and focus on Pressure + Pen Buttons.
  • For a "living stroke" style, start from the default and lower the minWidthFactor to 10%: you get very thin stroke starts and ends.
  • For technical drawing or lettering, use the Hard curve: low pressure writes very thin, medium pressure keeps the line stable.
  • Map the lower button to ↩️ Undo and the upper button to 🧼 Quick Eraser: it is the most convenient setup for drawing + quick correction without lifting the pen from the tablet.
  • If the pen buttons do not respond, first of all calibrate them: the fallback heuristic works on 70% of drivers, but calibrating unlocks them 100%.
  • If you need to modify the tablet active areas, system pressure, or orientation, use 🪟 Open Wacom Preferences: that is the official driver panel.
  • Disabling Wacom integration (master checkbox), the app behaves as with a mouse: all factors return to 1.0 and the tool sliders regain full effect.

Mouse & Keyboard Mapping

Mosaica's classic shortcuts (Ctrl+Z/C/V/X, Delete, Ctrl+/−, arrows, Ctrl+R, double-click) are protected and always active. The Input Mapping module lets you add on top of these:

  • Actions on single mouse buttons (left, middle/wheel, right, X1, X2)
  • Simultaneous mouse button combinations (chord)
  • Custom keyboard shortcuts (e.g. Ctrl+L = toggle Lasso)
  • Keys as directional arrows (e.g. W/A/S/D behaving as ↑/←/↓/→ — in parallel with native arrows)

Where to Find the Mapping

In the top bar at the top right, after the 🖊️ Wacom button, is the 🖱️⌨️ Input button. Click it to open the configuration modal, organized in four sections plus an advanced options block.

Section 1 — Single Mouse Buttons

For each MouseEvent.button code you can assign an action:

CodeNameDefault in Mosaica
0LeftNone (reserved for selection/drawing)
1Middle (wheel)None
2RightNone (reserved for fine rotation on shapes)
3X1 / Browser Back (side)↩️ Undo
4X2 / Browser Forward (side)↪️ Redo
On HP HSA-P007M mice, gaming mice, and many "browser back/forward" mice, the two side buttons emit button=3 and button=4. Mosaica intercepts these codes from mousedown, auxclick, and the virtual keys BrowserBack/BrowserForward: mappings work regardless of the driver.

🎯 Detect Key Code — discover what code your mouse sends

If you have a mouse with exotic buttons (thumb, DPI, programmable) and don't know what button code they emit, press "🎯 Detect key code": the yellow banner says "⏳ Press the mouse button you want to detect now…". Press the physical button, Mosaica captures the code and adds it to the list so you can assign an action from the dropdown. Safety timeout after 15 seconds.

Section 2 — Combinations (Chord)

Press "➕ Add combination" and hold down the mouse buttons you want to combine simultaneously (e.g. left + right). Release: after 0.5 seconds of idle, Mosaica registers the combination. Classic example:

  • Left + Right → Undo
  • Right + Middle → Zoom reset
  • X1 + X2 → Save project

Combinations have absolute priority over singles: as soon as the chord completes, the single action of the component button does not fire. Moreover, chords that include the right button block fine rotation of mouseObserver, so it does not trigger by mistake.

Each registered combination has three controls: the action menu on the right, the 🎯 Relearn button (to change the buttons without deleting the mapping), and ✕ to remove it.

Section 3 — Custom Keyboard Shortcuts

Press "➕ Add shortcut": the banner asks you to press the combination (modifiers + key). Examples:

  • Ctrl+L → Toggle Lasso (default already configured)
  • Alt+W → Toggle Watercolor
  • Ctrl+Shift+D → Duplicate selection
  • F2 → Toggle Pen
Override of hardcoded shortcuts — Mosaica's classic shortcuts (Ctrl+Z/C/V/X, Delete, Ctrl+/−, Ctrl+R, arrows) remain active by default. If you add a custom shortcut with the same combination, your version will have precedence (the override is explicit: you know what you are doing). Removing it, the hardcoded one comes back into effect.

The Ctrl and Cmd (on macOS) modifiers are treated as equivalent: a shortcut saved as "Ctrl+L" also matches "Cmd+L".

Section 4 — Keys as Directional Arrows (🎮)

Inside the 🎮 Keys as directional arrows section you find four buttons: ➕ Map to ↑, ➕ Map to ↓, ➕ Map to ←, ➕ Map to →. Press one, then press the physical key on the keyboard you want to associate with that direction. Classic gaming/CAD setup example:

  • W → ↑ (up)
  • S → ↓ (down)
  • A → ← (left)
  • D → → (right)
The native arrows (↑↓←→) always remain active in parallel: they are never disabled. Mapped keys are additional, so you can use both without choosing.

Movement step — identical to native arrows, calibrated on your monitor (uses mm2px from calibration):

ModifierStepWhen to use it
none0.1 mmDefault — fine precision for placing mosaic tiles
Shift1 mm"Large" step — quick movements
Ctrl0.01 mmSuper-fine sub-pixel — for perfect alignments
Priority over shortcuts — a key mapped as an arrow fires ONLY when pressed alone or with Shift. Combinations with Ctrl, Alt, or Cmd (e.g. Ctrl+W) remain free for custom keyboard shortcuts. Native arrows use Ctrl+arrow for the super-fine step; on mapped keys the same modifier works the same way.

Each binding has two controls: 🎯 Relearn key (to change the letter while keeping the direction) and ✕ to remove it. A key can be mapped to only one direction: attempting to map the same key twice shows a warning.

Without a shape selected, mapped keys perform a 5 mm view pan in the associated direction (exactly like native arrows).

⚙️ Advanced Options

Opening the "⚙️ Advanced Options" block at the bottom of the modal you find three fine adjustments:

Chord disambiguation delay (0–300 ms)

When you press a button that could be part of a mapped combination, Mosaica waits N milliseconds before firing the single, to give the combo time to form. Default 80 ms: balanced between responsiveness and chord precision. Set to 0 for maximum responsiveness (risk: the single may "double-fire" before the chord).

Preserve fine rotation on right button

If active (default ON), the single mapping on the right button does not fire when the pointer is over a rotatable shape: in that case mouseObserver's fine rotation (right button + wheel) takes precedence. On empty workspace areas the mapping activates normally.

Suspend in drawing mode

If active (default ON), when a drawing tool is active (Pen, Watercolor, Eraser) single mappings on the left button are suspended: the left button is used for drawing. Combinations and keyboard shortcuts continue to work normally.

Enable / Disable / Reset

  • The checkbox "Enable custom mappings" at the top globally disables the module: you return to standard behavior (left=selection, right=rotation on shapes). Useful as a "kill switch" if you want to compare behaviors.
  • "🔄 Restore defaults" at the bottom: resets all mappings to factory values (X1→Undo, X2→Redo, Ctrl+L→Lasso, no chords configured).
Mapping settings are saved in localStorage (key mosaica_input_mapping_settings) and persist across sessions. They are independent from .msp.json projects.

Practical Tips
  • If you have a mouse with side buttons, leave the defaults X1→Undo, X2→Redo: it is the most productive setup for mosaic design, where making mistakes and correcting is continuous.
  • A convenient combination is Left+Right → Deselect All: with two fingers you "turn off" any active object without having to click elsewhere.
  • For drawers who use the keyboard with the left hand: map F1 → Pen, F2 → Watercolor, F3 → Eraser, F4 → Lasso. Change tool without looking up.
  • If a shortcut does not work, check if you are inside a text field (Inspector, W/H dimensions, calibration): keyboard mappings are suspended when an input is focused — exactly like hardcoded shortcuts.

  • For debugging, open the DevTools console (Ctrl+Shift+I): the two modules print the loaded mappings at startup ([InputMapping] module initialized — prefs: … and [Wacom] Module initialized — prefs: …).

Security & Online Updates

Mosaica Workspace Pro is an offline drawing application: the only thing it does on the network is check whether a new version exists on the official GitHub repository and, at your request, download its installer. Everything else in the program never talks to the internet. This section explains how updates work and which protections are always active.

Checking and installing updates

Open the File menu in the top bar and click 🔄 Check for updates. Mosaica queries the official GitHub releases and opens the "Mosaica Updates" modal, always showing the result:

  • You are up to date → message "You are using the latest version of Mosaica Workspace Pro".
  • A new version is available → you see the current version, the new version, the release notes and the installer name/size. From here you can Download and install, postpone with Later, or Skip this version (it will no longer be proposed in automatic checks until a newer one is released).

During the download a progress bar shows percentage and bytes; you can keep working and, if you wish, press Cancel. Once the download is finished choose Install now (Mosaica closes and launches the installer) or Install later.

When you choose Install now, auto-save runs before closing: if you have unsaved work, the "Saving…" overlay secures it before the installer starts. You lose nothing.

Automatic check at startup

If the "Check at startup" box is enabled, a few seconds after opening Mosaica performs a silent check: it opens the modal only if it actually finds a new (and not "skipped") version. The automatic check is limited to no more than once every 6 hours, so it never gets in the way. You can disable it by unchecking the box in the modal.

⚠️
Automatic updates work only on the installed Windows version. In development mode (npm run dev) or on other platforms Mosaica simply tells you so — it does not download anything.

How Mosaica protects the download

Downloading and running an installer is the most delicate moment: that is why the update goes through a series of purpose-built security checks before any file is launched.

🔒 GitHub only

The download is allowed only towards the official GitHub domains (HTTPS). Any redirect is re-checked: if it points to a foreign host, it is blocked and aborted.

🧾 Verified filename

The installer must have a simple name with the .exe extension, with no path separators and no .. (no "path traversal").

🧬 File integrity

After the download Mosaica checks the size (within sensible limits: 1–500 MB), that the file is really a Windows executable ("MZ" magic bytes) and compares its SHA-256 hash with the one published by GitHub. If the hash does not match, the file is deleted and the installation refused.

🗒️ Security log

Relevant events are recorded in %APPDATA%\mosaica-workspace-pro\security.log (with automatic rotation). It is only for diagnostics: no data leaves your PC.

Older releases may not have the SHA-256 hash: in that case the size and file-type checks remain active, and the event is recorded in the log. It is one fewer check, not an absence of checks.

Always-on defenses (even offline)

Regardless of updates, Mosaica "locks itself down" at startup so it stays what it is meant to be: a local editor.

  • No web navigation: the window cannot be taken to external pages; http(s) links (e.g. the project page) open in the system browser, never inside the app. No popups, no nested windows.
  • Permissions denied by principle: camera, microphone, geolocation, notifications and the like are always refused — Mosaica does not need them.
  • Strict Content-Security-Policy: the program loads only its own local resources and never runs code coming from the network. The renderer never talks to the internet directly: every network operation goes through the main process.
  • Independent of the drawing engine: this security layer does not touch Fabric and is fully compatible with Fabric.js from v5.1.0 to v5.3.0.
⚠️
On the first launch after an update, Windows Defender may show the "Unknown publisher" warning because the installer is not signed with a paid certificate. This is normal: click More info → Run anyway. The SHA-256 verification done by Mosaica still guarantees that the downloaded file is identical to the published one.
🎯
Practical tips
  • Leave "Check at startup" on: the check is light, silent and notifies you only when something genuinely new is released.
  • If a version does not interest you, use Skip this version: automatic checks will ignore it until a later one is released.
  • You can keep drawing during the download: installation starts only when you press Install now, and auto-save protects your work anyway.
  • If you are offline or GitHub does not respond, the manual check tells you clearly and the app keeps working normally offline.
